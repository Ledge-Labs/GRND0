# Changelog

## 0.0.1 - 2026-07-20

- Released OpenAI chat-completions and Anthropic messages contracts with
  client-controlled structured tool calls.
- Shipped the conversation-grade loop: subject resolution, typed
  comprehension, effort-aware routing, connected retrieval, recursive context
  reduction, aperture-aware synthesis, verification, typed abstention, one
  discourse commit, and durable execution receipts.
- Added report-scale outline, per-section authoring over one bounded working
  set, deterministic folding, and single-call degradation on structured-path
  failure.
- Bundled digest-pinned llama-swap integration, an untracked model store,
  verified model download, deterministic serving configuration, and per-lane
  local or explicitly granted cloud backends.
- Added the extracted native Windows Vulkan serving path for local GPU lanes,
  with generated host configuration and explicit backend grants.
- Added the extracted persistent WSL model-process boundary and local backend
  wiring for model runtimes that require Linux.
- Added a SQLite-only core with public self-knowledge and optional web, local
  Gitea, folder, local-index, and vector retrieval capabilities under explicit
  grants.
- Defined core, web, git, and full capability profiles; every published port
  remains loopback-bound by default and fixed credentials are absent.
- Published the immutable reference chat pack, pending-result schema,
  manifest-driven deterministic exporter, provenance ledger, SBOM, notices,
  and fail-closed leak gate.
